Sobre o Projeto

Documentação completa do site LivroMania — objetivos, tecnologias, público-alvo e decisões de design.

Objetivos

Criar uma livraria virtual completa e responsiva que ofereça uma experiência de compra agradável e intuitiva para leitores de todos os perfis.

  • Exibir catálogo com lançamentos e mais vendidos
  • Oferecer canal de atendimento via formulário de contato
  • Permitir Cadastro
  • Organizar acervo em gêneros literários

Público alvo

O site é voltado para leitores brasileiros de qualquer faixa etária que buscam adquirir livros online com comodidade e variedade. Engloba desde jovens leitores iniciantes até colecionadores experientes, passando por estudantes, professores e profissionais de diversas áreas.

Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando HTML5 e CSS3:

  • HTML5 - Responsável por toda a estrutura e arquitetura semântica do site.

  • CSS3 - Responsável por todas as estilizações de cores, tipografia e layout. O design foi planejado para ser responsivo, adaptando-se a qualquer tamanho de tela.

Restrições e Requisitos Mínimos

Para o usuário

  • Resolução mínima: 320 px
  • Conexão à internet para carregamento de imagens

Restrições do projeto

  • Backend e autenticação não implementados
  • Carrinho de compras é demonstrativo (sem integração de pagamento)

Dificuldades Encontradas

Dificuldades

  • Construção do Menu Hamburguer
  • Resposividade das imagens de capas para um design harmônico
  • Organização de páginas e links

Pontos Fortes e Fracos

Pontos fortes

  • Design responsivo completo (mobile, tablet e desktop)
  • Navegação clara
  • Paleta de cores coerente

Pontos Fracos

  • Ausência de back-end real
  • Sem sistema de busca funcional
  • Sem integração com plataforma de pagamento

Periodicidade de Atualização

Este site é um projeto estático, desenvolvido exclusivamente com HTML5 e CSS3. Isso significa que ele não possui automação de dados em tempo real.

As atualizações de conteúdo, melhorias de design e ajustes de código são feitas de forma manual e periódica, garantindo que as informações apresentadas estejam sempre revisadas e organizadas.
